# 俄罗斯方块

### 一、环境依赖
c++14

### 二、程序运行
1. 拉取代码
```shell
git clone git@github.com:Liaoshaoyu/myTinytetris.git
```
2. 安装curses
```shell
sudo apt-get install libncurses5-dev 
```
3. 构建
```shell
g++ -o main main.cpp -lncurses
```
4. 运行
```shell
./main
```
